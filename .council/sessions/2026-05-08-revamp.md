# Council session: Quiet Ledger full-site revamp

## Why convened

Chairman directive: "The website needs a complete revamp. The council
agents." Phase A from `.council/sessions/2026-05-08-rollout.md` widens
into a multi-chunk rebuild owned by Technical Lead with section editors
reviewing copy on the routes inside their beats.

The `.dp-*` token retune from PR #17 (May 2026) already re-themes every
page that consumes those classes — card reviews, guides via
`GuideLayout`, bank hubs, airline programmes, valuations. The remaining
work is the routes that still carry **scoped CSS or Tailwind utilities**
that pre-date the system or never adopted it.

## Scope

In priority order. Each chunk is its own branch + PR.

### Chunk 1 — Homepage rebuild (this session)

- File: `src/pages/index.astro` — currently 668 lines, ~427 of which
  are scoped CSS spanning 7 sections.
- Target: 4-section Quiet Ledger spec from
  `.council/research/2026-05/ux-redesign-brief.md`:
  1. Masthead-dateline + 5-story typographic index (no images)
  2. Live: salary transfer + deals rail (combined two-column module)
  3. Latest analysis (single-column reverse-chronological feed)
  4. Newsletter + RSS-by-band
- Owners by stage:
  - Stage 4 SEO Pass: SEO Strategist (post-rebuild review)
  - Stage 5 Draft: Technical Lead (this commit)
  - Stage 7 Tech Pass: Technical Lead (self-validate)
  - Stage 8 Chairman Gate: user
- Deal-rail negative-margin bug at the old `index.astro:163-177` dies
  by collapsing the rail into the Live module.
- Acceptance:
  - `npm run check` clean
  - `npm run build` succeeds; Lighthouse mobile ≥ 95
  - 8 sections become 4 visibly
  - Gold appears nowhere on the homepage
  - Existing data sources preserved (guides + deals + cards +
    salaryTransferOffers collections)
- Branch: `claude/quiet-ledger-revamp` (this branch)

### Chunk 2 — Trust pages

- Files: `src/pages/about.astro`, `src/pages/team.astro`,
  `src/pages/editorial-policy.astro`,
  `src/pages/how-we-make-money.astro`.
- Audit each for Tailwind `prose` classes and scoped CSS that should
  be migrated to `.dp-*`. These are short pages — fast wins.
- Owners: Technical Lead writes; Chairman reviews voice on
  `editorial-policy` and `how-we-make-money`.
- Branch: `claude/quiet-ledger-trust-pages`

### Chunk 3 — Directory pages

- Files: `src/pages/cards/index.astro` and the category subpages
  (`cashback.astro`, `islamic.astro`, `miles.astro`, `compare.astro`),
  `src/pages/banks/index.astro`, `src/pages/airlines/index.astro`,
  `src/pages/guides/index.astro`.
- These should mostly inherit cleanly from `.dp-dir-grid` and
  `.dp-dir-tile`. Audit + fix any that don't. Verify card grid
  scales to ~40 entries cleanly (the design-spike test).
- Owners: Technical Lead writes; section editors review their own
  vertical's directory.
- Branch: `claude/quiet-ledger-directories`

### Chunk 4 — Salary-transfer pages

- Files: `src/pages/salary-transfer/index.astro`,
  `src/pages/salary-transfer/calculator.astro`,
  `src/pages/salary-transfer/[slug].astro`,
  `src/pages/salary-transfer/history/[bank].astro`. The Preact island
  `SalaryTransferTracker.tsx` already adopts `.dp-tracker-*` styles
  and inherits the retune for free; spot-check.
- Owners: Technical Lead writes; Business & Real Estate Editor
  reviews copy.
- Branch: `claude/quiet-ledger-salary-transfer`

### Chunk 5 — Valuations pages

- Files: `src/pages/valuations/index.astro`,
  `src/pages/valuations/methodology.astro`. Methodology is currently
  `editorial-policy`-style prose; migrate to `.dp-prose`.
- Owners: Technical Lead writes; Chairman gates methodology copy
  (load-bearing trust artefact).
- Branch: `claude/quiet-ledger-valuations`

## Out of scope for this revamp session

- Adding new routes (per `.council/04_content_taxonomy.md`) — that
  is Phase B from the earlier rollout plan, queued separately.
- Header redesign — Header already adopts the continuous logo
  scale (36/44/52); no further change indicated by the brief.
- Footer redesign — same rationale; the Footer already lives in
  the `.dp-*` system.
- Bank scrapers (ADCB, Mashreq, etc.) — Phase C, separate cadence.
