# Council session: TPG-inspired visual pivot

## Why convened

Chairman directive: "I want my website to look like https://thepointsguy.com.
Can you scrape the layout, site design and colour coding. Can the council
look at it and replicate. I hate the current look of my site. We need to
finalise the design parameters site structure to start putting stories
and news."

Net: the previously-adopted "Quiet Ledger" direction (May 2026 brief,
PR #17) is **overridden**. The new visual reference is The Points Guy,
a US-based points-and-miles publication.

## What this pivot does AND does not change

### Visual scope (in scope)

- Palette, type, and component idiom across the whole site
- Homepage IA / section structure
- Card review template
- Directory and category landing pages
- Trust pages (`about`, `team`, `editorial-policy`, `how-we-make-money`)
- Salary-transfer pages and the live tracker visual
- Valuations pages
- `src/styles/global.css` token set and `.dp-*` component definitions

### Editorial fence (out of scope — NON-NEGOTIABLE)

The Charter's editorial discipline does **not** weaken because the
visual reference shifts. The following remain in force regardless of
which palette / type / layout we end up with:

- **AED-first pricing** on every page; USD only in parentheses on
  international reads (`01_editorial_standards.md` §1)
- **No advertorial-driven recommendations** at launch (`CLAUDE.md`
  Part I non-negotiable §5)
- **Affiliate disclosure inline above the fold** when it eventually
  lands; never a sponsor pin above the fold
- **Chairman publish gate** (`02_workflow.md` Stage 8); no piece
  ships without `chairman-status: approved`
- **Firecrawl exclusive to Head of Research**; SEO Strategist routes
  via `Task` (Charter non-negotiable §2)
- **LLM-extraction policy** — Firecrawl `/extract` is allowed for
  prose seeding, off-limits for typed numerics; deterministic regex
  for fees, salary bands, earn rates, welcome-bonus amounts
  (Charter §6)
- **HfP-dry voice** in the prose — TPG visual does **not** mean TPG
  voice; the editorial standards in `01_editorial_standards.md` §6
  carry through unchanged

If the TPG-adapted brief proposes a visual pattern that conflicts with
any of the above, the Charter wins and the brief amends.
